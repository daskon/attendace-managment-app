## About <a name = "about"></a>

Attendance tracking app has developed using latest version of Reactjs 18 along with Laravel back-end.

### Installing

```
cd app
npm install
npm start
```
```
cd api
composer install
php artisan migrate
php artisan serve
```

```
Front page : http://127.0.0.1:3000
Front page : http://127.0.0.1:3000/tasktwo
```
